# Airbnb sauna finder

Airbnb is a bad service without the possibility of filtering by sauna. This script should fix it.

## Use

1. Install dependencies

```
yarn
```

2. Set `editableParams` (index.js)

3. Run script

```
yarn start
```

## Info

Open links in Terminal (Mac): `CMD + [double click]`
